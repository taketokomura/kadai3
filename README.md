ロボットシステム学追加課題

## 動作環境
OS:Ubuntu 18.04

## インストール方法
```
$ git clone https://github.com/taketokomura/kadai3.git
```

## 実装機能
```
./kadai3 計算したい整数を入力後自動で素因数分解をしてくれる。
```
## 使用方法
```
$ gcc kadai3.c -o kadai3 -rm コンパイル
$ ./kadai3　⇒　計算したい数値を入力
```
