## 動作環境
OS:Ubuntu 18.04

## インストール方法と使用方法
```
$ git clone https://github.com/taketokomura/kadai3.git
$ cd kadai3   
$ gcc kadai3.c -lm -o kadai3
$ ./kadai3
```

## 実装機能
```
./kadai3 計算したい整数を入力後自動で素因数分解をしてくれる。
```
